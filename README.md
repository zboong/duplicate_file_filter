# JWFileFilter

사진과 동영상 파일을 관리하고 중복 파일을 찾아 정리하는 Python GUI 프로그램입니다.

## 기능

- **중복 파일 검색**: 파일 크기와 수정 시간을 비교하여 중복 가능성이 높은 파일 쌍을 찾아냅니다.
- **이미지 뷰어**: 중복으로 의심되는 이미지 파일들을 나란히 배치하여 시각적으로 비교할 수 있습니다.
- **파일 필터링**: 이미지 파일(.jpg, .png 등), 동영상 파일(.mp4, .mov 등)을 선택적으로 검색할 수 있습니다.
- **하위 폴더 포함**: 재귀적 검색 옵션을 통해 하위 디렉토리까지 모두 검사할 수 있습니다.
- **간편한 삭제**: 비교 후 불필요한 파일을 즉시 삭제하여 저장 공간을 확보할 수 있습니다.

## 지원 파일 형식

### 이미지 파일
- JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP

### 동영상 파일  
- MP4, AVI, MOV, WMV, FLV, MKV, WEBM

## 설치 및 실행

### 요구사항
- Python 3.8 이상
- Pillow (이미지 처리 라이브러리)

### 환경 설정 (venv 사용 권장)
```bash
# 가상 환경 생성
python -m venv venv

# 가상 환경 활성화 (Windows)
.\venv\Scripts\activate

# 필요 패키지 설치
pip install -r requirements.txt
```

### 실행 방법
```bash
python jwfilefiltergui.py
```

## 사용법

1. **디렉토리 선택**: "찾아보기" 버튼을 클릭하여 중복 파일을 검사할 폴더를 선택합니다.
2. **필터 및 옵션**: 검색할 파일 유형과 하위 폴더 포함 여부를 선택합니다.
3. **중복 파일 검색**: "중복 파일 검색" 버튼을 클릭합니다.
4. **비교 및 삭제**: 
   - 목록에서 중복 쌍을 선택하면 상단에 이미지가 표시됩니다.
   - 삭제할 파일 아래의 체크박스를 선택한 후 "선택된 파일 삭제" 버튼을 클릭합니다.
